## 0.0.2

* Renamed package from `orb` to `siri_orb`.
* Added `SiriORB` widget — an authentic iOS 14+ Siri-style glowing sphere using `BlendMode.plus` additive color layering with wavy organic petal ribbons.
* Added `ImageFilter.blur` composite to `SiriORB` for a premium soft-glow appearance.
* Fixed infinite animation loop jitter by enforcing integer time multipliers on all ribbon rotations.
* Improved audio amplitude reactivity with smooth `lerpDouble` interpolation.
* Updated README with full API reference, comparison table, and platform badges.
